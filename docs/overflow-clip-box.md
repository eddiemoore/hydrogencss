# overflow-clip-box

https://developer.mozilla.org/en-US/docs/Web/CSS/overflow-clip-box

## content-box
```css
.selector {
  composes: content-box from 'hydrogencss/overflow-clip-box.css'
}
```

or:
```css
.selector {
  composes: content-box from 'hydrogencss/overflow-clip-box/content-box.css'
}
```
