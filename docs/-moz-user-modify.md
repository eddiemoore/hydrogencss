# -moz-user-modify

https://developer.mozilla.org/en-US/docs/Web/CSS/-moz-user-modify

## read-only
```css
.selector {
  composes: read-only from 'hydrogencss/-moz-user-modify.css'
}
```

or:
```css
.selector {
  composes: read-only from 'hydrogencss/-moz-user-modify/read-only.css'
}
```
