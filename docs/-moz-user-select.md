# -moz-user-select

https://developer.mozilla.org/en-US/docs/Web/CSS/-moz-user-select

## -moz-none
```css
.selector {
  composes: -moz-none from 'hydrogencss/-moz-user-select.css'
}
```

or:
```css
.selector {
  composes: -moz-none from 'hydrogencss/-moz-user-select/-moz-none.css'
}
```
