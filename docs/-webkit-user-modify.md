# -webkit-user-modify

https://developer.mozilla.org/en-US/docs/Web/CSS/-webkit-user-modify

## read-only
```css
.selector {
  composes: read-only from 'hydrogencss/-webkit-user-modify.css'
}
```

or:
```css
.selector {
  composes: read-only from 'hydrogencss/-webkit-user-modify/read-only.css'
}
```

## read-write

.selector {

.selector {
