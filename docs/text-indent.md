# text-indent

https://developer.mozilla.org/en-US/docs/Web/CSS/text-indent

## each-line
```css
.selector {
  composes: each-line from 'hydrogencss/text-indent.css'
}
```

or:
```css
.selector {
  composes: each-line from 'hydrogencss/text-indent/each-line.css'
}
```

.selector {
