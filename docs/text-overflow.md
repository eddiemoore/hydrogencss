# text-overflow

https://developer.mozilla.org/en-US/docs/Web/CSS/text-overflow

## clip
```css
.selector {
  composes: clip from 'hydrogencss/text-overflow.css'
}
```

or:
```css
.selector {
  composes: clip from 'hydrogencss/text-overflow/clip.css'
}
```

```css
.selector {

}
```

or:

```css
.selector {
```
