# appearance

https://developer.mozilla.org/en-US/docs/Web/CSS/appearance

## auto
```css
.selector {
  composes: auto from 'hydrogencss/appearance.css'
}
```

or:
```css
.selector {
  composes: auto from 'hydrogencss/appearance/auto.css'
}
```

.selector {
