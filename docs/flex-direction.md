# flex-direction

https://developer.mozilla.org/en-US/docs/Web/CSS/flex-direction

## column
```css
.selector {
  composes: column from 'hydrogencss/flex-direction.css'
}
```

or:
```css
.selector {
  composes: column from 'hydrogencss/flex-direction/column.css'
}
```
