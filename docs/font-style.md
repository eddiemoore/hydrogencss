# font-style

https://developer.mozilla.org/en-US/docs/Web/CSS/font-style

## italic
```css
.selector {
  composes: italic from 'hydrogencss/font-style.css'
}
```

or:
```css
.selector {
  composes: italic from 'hydrogencss/font-style/italic.css'
}
```
