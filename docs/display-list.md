# display-list

https://developer.mozilla.org/en-US/docs/Web/CSS/display-list

## list-item
```css
.selector {
  composes: list-item from 'hydrogencss/display-list.css'
}
```

or:
```css
.selector {
  composes: list-item from 'hydrogencss/display-list/list-item.css'
}
```

```css
}
```

or:
```css
.selector {

}
```
