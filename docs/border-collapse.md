# border-collapse

https://developer.mozilla.org/en-US/docs/Web/CSS/border-collapse

## collapse
```css
.selector {
  composes: collapse from 'hydrogencss/border-collapse.css'
}
```

or:
```css
.selector {
  composes: collapse from 'hydrogencss/border-collapse/collapse.css'
}
```

```css
.selector {

}
```

or:
```css
.selector {

```
